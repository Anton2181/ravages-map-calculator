Army

Army length affects march cost. Length > 6 mi doubles all land weights (the column drags). Fording a thin river costs length × 2.4 IRL hours per crossing (overrides the Fording weight when length > 0). Can ford off blocks thin-river crossings entirely; Can embark off blocks boarding ships.

Traversal weights

Each hex you enter costs its terrain weight. Lake / Sea / Ocean weights apply when sailing water-to-water. Embark / disembark is added each time the route crosses the shoreline. If the destination hex is flagged as a Road in the sheet, the Road column value replaces the default for that hex. Starting hex is free. Path recomputes on every change.
